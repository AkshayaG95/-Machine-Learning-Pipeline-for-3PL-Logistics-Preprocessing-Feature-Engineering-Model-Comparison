# Predictive Risk Modeling: ML Pipeline for Operational Decision-Making

> Built an end-to-end machine learning pipeline focusing on data preprocessing, feature engineering, and model evaluation to predict risk levels in operational systems.

---

## Overview
This project develops a machine learning solution to predict risk levels using structured operational data. The focus is on building a reliable pipeline that transforms raw data into actionable insights.

---

## Problem
Operational datasets are often noisy, imbalanced, and complex.

## Goal:    
Build a robust ML pipeline to accurately detect high-risk cases and support proactive decision-making.

---

## My Contributions
- Led data preprocessing and feature engineering
- Built the model comparison and evaluation framework  
- Designed approaches for class imbalance handling and threshold tuning  
- Contributed to interpretation of results  

---

## Approach
- Data cleaning and preprocessing  
- Feature engineering  
- Model training (Logistic Regression, SVC, Random Forest, XGBoost)  
- Model comparison using F1-score and ROC-AUC  
- Threshold optimization for rare event detection  

---

## Key Insights
- Preprocessing and feature engineering strongly impact performance  
- Accuracy is misleading in imbalanced datasets  
- Threshold tuning improves detection of rare events  
- Tree-based models perform best in complex data  

---

## Key Takeaway
> Reliable ML systems are built on strong data foundations and thoughtful evaluation, not just model choice.
